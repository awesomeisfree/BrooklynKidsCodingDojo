Have fun with your name

In this project, we're going to have fun using your name. (The purpose of this project is for students to practice simple effect blocks).

1) Add one character, make the character's color change when this character is clicked.

2) Add second character, make second character turn.

3) Add third character, click third character and play a sound.

4) Add fourth character, make fourth character move right, play a sound, move left and play a sound again.

5) Add fifth character, make fifth character enlarge and shrink.

6) Make sixth character change color and turn. (Student practice)

7) Make seventh sprite glide to the corners of the screen.

8) Make an interactive name card for your parents using what we learned today
